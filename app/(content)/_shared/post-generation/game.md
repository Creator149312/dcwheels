## 1. Gameplay & Mechanics Debates
For players who care about how a game feels to play.

- "Was the difficulty fair or punishing?" → Fair / A little harsh / Way too hard / Too easy
- "Did the controls feel natural?" → Perfect / Took getting used to / Never clicked / Ruined the experience
- "Is the game too long, too short, or just right?" → Way too long / Perfect length / Could've been longer / Depends on the playthrough
- "Did the game respect your time?" → Yes, very respectful / Mostly / Had unnecessary padding / No, it wasted my time
- "Open world or linear — which fit this game better?" → Open world was great / Should've been linear / The linearity was refreshing / Hybrid approach was best

---

## 2. Story & Narrative
For story-driven games and RPGs.

- "Did the story land for you?" → Loved every bit / Good but not great / Lost me halfway / Story wasn't the point
- "Was the ending satisfying?" → Perfect / It was okay / Felt rushed / Completely missed the mark
- "Best character in the game?" → list of main characters
- "Did you care about the protagonist?" → Deeply / A little / Not really / Actively disliked them
- "Was the villain compelling?" → One of the best / Had potential / Fell flat / Didn't need one
- "What if [major story choice] went differently?" → speculative / emotional angle

---

## 3. Value & Replay
For gauging whether a game is worth the price and time.

- "Was this game worth full price?" → Absolutely / Mostly / Should've waited for a sale / No
- "Would you replay it?" → Already have / Definitely will / Maybe someday / Once was enough
- "DLC: essential or cash grab?" → Essential / Worth it / Felt unfinished without it / Pure cash grab
- "How does it hold up today?" → Better than ever / Still great / Shows its age / Hasn't aged well
- "Would you recommend it to someone who doesn't play games often?" → Yes, great entry point / Only if they're into [genre] / Not really / Definitely not

---

## 4. Comparison & Legacy
For placing the game in the bigger picture.

- "Best game in its genre?" → Definitely / One of the best / Mid-tier / There are better options
- "Did this game change gaming in any way?" → Massively / A little / Not really / It was just fun
- "Sequel hype: do you want one?" → Desperately / Would be interesting / Leave it as it is / Please don't
- "Does it deserve its reputation (cult classic / masterpiece / overrated)?" → 100% deserved / Partially / Overhyped / Underhyped
- "Better than [comparable game in the same genre]?" → Yes easily / About the same / Not quite / No comparison

---

## 5. Player Identity & Choices
For games with meaningful decisions or playstyle variety.

- "What type of player were you?" → Explorer / Completionist / Speedrunner / Story-only / Chaos agent
- "Which ending did you get?" → Best ending / Bad ending / Multiple endings / There was only one?
- "Did you play it safe or take risks?" → Played it safe / Mix of both / All-in on risky choices
- "Which faction/class/path did you choose?" → faction/class options specific to the game
- "Did you read the lore or skip it?" → Read everything / Skimmed it / Skipped entirely / Didn't know there was lore

---

## 6. First Impression vs Hindsight
Captures both new players and veterans.

- "Did the game hook you from the start?" → Instantly / After a few hours / Took a long time / Never really did
- "Did your opinion change after finishing it?" → Better after finishing / Same / Worse after finishing / Still playing
- "Biggest surprise about this game?" → Better than expected / Worse than expected / The story caught me off guard / The difficulty
- "Wish you could play it for the first time again?" → Absolutely / Kind of / Not really / It'd be different without the hype

---

## Generation Rules for AI Prompt (Game)
- Use Gameplay & Mechanics for action, platformer, and competitive games.
- Use Story & Narrative for RPGs, visual novels, and narrative-driven games.
- Use Value & Replay for any game that was controversial in price or length at launch.
- Use Player Identity for games with branching paths, multiple classes, or factions.
- Reference specific mechanics, characters, DLCs, or sequels by name when possible.
- Favor 3-4 option polls over binary for games — players have nuanced opinions.
