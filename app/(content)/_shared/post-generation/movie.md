# Post Generation Strategy: Movie TopicPages
# Used by: scripts/ai-topic-post-bot.mjs

---

## 1. Philosophical & Moral Dilemmas
Great for thought-provoking or slow-burn films.

- "Was [protagonist]'s decision justified?" → Yes / No / It's complicated
- "Would you have made the same choice as [character]?" → Definitely / Never / Hard to say
- "Does the movie glorify something it shouldn't?" → Yes, it's a problem / No, it's nuanced / Depends on the viewer
- "Was the villain actually wrong?" → Completely wrong / Had a point / Was right all along
- "Should [character] have sacrificed themselves?" → Yes, it was necessary / No, there was another way / I'm still not over it

---

## 2. Experience & Watchability
How the film landed emotionally and practically.

- "Is this a movie you can rewatch?" → Many times / Once more maybe / Once was enough / Never again
- "Did the ending satisfy you?" → Loved it / It was okay / Felt rushed / Completely ruined it
- "Should this have been a sequel / franchise or stand-alone?" → Stand-alone was perfect / I'd watch a sequel / The sequels ruined it
- "Best format to watch this?" → Cinema / Home with friends / Solo late night / Doesn't matter
- "Would you recommend this to someone who doesn't watch this genre?" → Yes / No / Only if they're ready for it

---

## 3. Casting & Craft Debates
For cinephiles and casual viewers alike.

- "Was the casting perfect?" → Spot on / Some roles were off / One actor carried it / Complete miscast
- "Did the CGI/visuals help or hurt the film?" → Made it iconic / Was fine / Aged badly / Took me out of it
- "Was the soundtrack one of the best parts?" → Absolutely / It was okay / Barely noticed it / It didn't fit
- "Too long, too short, or just right?" → Could've been shorter / Perfect runtime / Needed more time
- "Director's best work?" → Yes easily / One of the better ones / Not their best / Worst one

---

## 4. Cultural Impact & Debate
For films that sparked conversation beyond the screen.

- "Did this movie deserve its awards?" → 100% / Partially / Was robbed / Was overrated
- "Is this film overrated or underrated?" → Criminally underrated / Rightly rated / Overhyped
- "Did this change how you see a certain topic?" → Completely / A little / Not really / Made it worse
- "Is this a 'must watch once in your life' film?" → Absolutely / Depends on the person / Not really
- "Will this be remembered in 20 years?" → Classic for sure / Probably not / Already forgotten

---

## 5. First Impression vs Hindsight
For people mid-watch, done, or rewatching.

- "Did you go in with high expectations?" → Yes and it delivered / Yes and it disappointed / No expectations and loved it / No expectations and meh
- "Does it get better on a rewatch?" → Way better / Same / Not as good / Haven't rewatched
- "What did you get wrong about this film before watching?" → Everything / A few things / Nothing, I knew / Still haven't watched
- "Did the trailers spoil too much?" → Yes, ruined moments / A little / No, trailers were perfect

---

## 6. Just-for-Fun / Light Engagement
Low-effort, high-participation polls.

- "Which character would you be?" → list of main characters
- "Which scene do you wish you could erase from your memory?" → key emotional/shocking scenes
- "Watch it dubbed or subbed?" (if foreign film) → Dubbed / Subbed / Either works
- "Cinema or streaming?" → Had to be in cinema / Streaming was fine / Wish I saw it in cinema

---

## Generation Rules for AI Prompt (Movie)
- Prioritize Craft & Casting debates for big-budget blockbusters (Marvel, DC, big franchises).
- Prioritize Philosophical questions for drama, thriller, or sci-fi films.
- Use Cultural Impact category for award-winning or critically polarizing films.
- Use Just-for-Fun for light comedies or franchise films to keep it playful.
- Keep poll options to 3-4 max. Avoid binary yes/no unless the question is truly a dilemma.
- Reference specific characters, directors, or scenes by name when possible.
