# Post Generation Strategy: Anime TopicPages
# Used by: scripts/ai-topic-post-bot.mjs

---

## 1. Philosophical & Moral Dilemmas
Force users to confront the big ethical questions anime explores.

- "Would you support [protagonist's controversial plan]?" → Yes / No / It's complicated
- "Do the ends justify the means?" (e.g. Code Geass) → Always / Never / Only if the cost is low
- "Should heroes kill villains?" (e.g. My Hero Academia) → Yes / No / Case-by-case
- "Would you take the Death Note?" → Absolutely / Never / Only if anonymous
- "Is pain necessary for growth?" (e.g. Naruto) → Yes / No / Depends on the person

---

## 2. Identity & Role Questions
Let users imagine themselves inside the anime world.

- "Which role would you play in this anime?" → Hero / Sidekick / Mentor / Villain / Bystander
- "Would you survive in this universe?" → Definitely / Maybe / No chance
- "Which faction would you join?" (e.g. Akatsuki vs Konoha) → faction-based options
- "Would you rather train under [Character A] or [Character B]?" → just-for-fun loyalty test
- "What power from this show would you take?" → show-specific ability list (e.g. Nen types)

---

## 3. Reaction & Emotional Engagement
Surface iconic moments and emotional memory.

- "What was the most emotional moment in [Anime]?" → open discussion / comment driver
- "Who carried the show?" → character name options (great for surfacing side characters)
- "What's the best dialogue/quote?" → resurface iconic lines
- "What emotion did this anime leave you with?" → Joy / Sadness / Inspiration / Anger / Emptiness
- "Which moment hit you the hardest?" → drives comments and personal stories

---

## 4. Discussion Starters / Debates
For morally grey and highly debated anime.

- "Underrated character in [Anime]?" → highlight background/support characters
- "Was the ending good?" → Yes / No / Rushed / Masterpiece / Didn't finish it
- "Who was right?" → for morally grey shows (e.g. Eren vs the world)
- "What if [key character] didn't die?" → speculative/emotional angle
- "Did this anime change your view on something?" → depth and personal reflection
- "Was [Character] a good partner/leader/villain?" (e.g. Zero Two) → character loyalty debate

---

## 5. Deeper Viewer Behavior (Profiling)
Great for understanding the audience and surfacing patterns.

- "What keeps you watching?" → Characters / Story / Animation / Music
- **Pre-watch:** "Worth starting?" → targets non-watchers. Great for newer catalog pages.
- **Post-watch:** "What did this anime leave you with?" → targets fans. Better for highly-rated shows.
- "Loyalty test: Would you have sided with [Character] in [Arc]?" → e.g. Eren in Season 4, Light vs L
- "Romance take: Was [ship] a good pairing?" → triggers strong opinions from shippers
- "Would you rewatch [Anime]?" → Already have / Maybe someday / Once was enough / Couldn't finish
- "Would you recommend it to a non-anime fan?" → forces nuance and filters out bad entry points

---

## 6. First Impression vs Hindsight
Captures re-watchers, lurkers, and people mid-way through a show.

- "When you started [Anime], did you expect to love it?" → Hated the start, loved it later / Loved it from ep 1 / Still not sure how I feel / Dropped it tbh
- "Did [Anime] get better or worse as it went on?" → Got way better / Stayed consistent / Started strong, fell off / Still watching
- "Does [Anime] hold up on a rewatch?" → Even better / Same experience / Not as good / Haven't rewatched
- "Were your expectations met?" → Exceeded / Met / Disappointed / Going in blind was the right call

---

## Generation Rules for AI Prompt (Anime)
- Always reference specific show mechanics, arcs, or character names when possible.
- Favor binary or 3-option polls for maximum engagement (not more than 4 options).
- Use 1 of the 5 categories per post — don't mix categories in one post.
- Keep content lowercase and casual tone. Avoid heavy slang.
- Use debate/moral angle when the show is well-known (AoT, Naruto, MHA, Code Geass, SAO).
- Use identity/role angle for niche/lesser-known shows to draw in curious non-watchers.
- The AI should pick pre-watch vs post-watch based on the TopicPage's community rating.
